Graphics Pipeline Basics

Windowing

Windowing is the process of creating and managing a window that displays graphics and receives input. It also supports event polling and buffer swapping, which are essential for handling window interactions and smoothly displaying rendered frames.

Buffer Objects

Buffers store data like vertices, colors, and indices on the GPU. They are essential for passing geometry and other information to shaders for rendering.

Draw Call

A Draw Call is the process of issuing instructions to the GPU to render specific objects.

Shaders

Shaders are often small GPU programs that control how objects are drawn. They process vertex data and determine the final color and appearance of each pixel.

Depth Testing

Depth testing ensures that pixels closer to the camera are drawn over those farther away, maintaining proper object overlap and realistic 3D rendering.

Textures

Textures are images applied to 3D objects to add surface detail and color. They make objects look more realistic by giving them details and effects without adding extra geometry.

Camera

The camera defines the viewer's perspective in a 3D scene. It controls the field of view, position, orientation, and how the scene is projected onto the screen.

FrameBuffers

Framebuffer's allow rendering to off-screen targets. They're used for post-processing effects, rendering textures, and are used in more advanced graphics pipelines.

Render Passes

A render pass is a single step in the graphics pipeline where the scene is drawn or processed in a specific way. Usually multiple passes happen, each responsible for different tasks.

Windowing

What is Windowing?

Windowing is the process of creating and managing an operating system window that serves as the surface for rendering graphics and receiving user input.

System APIs

It involves interacting with the OS's window manager through APIs like Win32, X11, Wayland, or Cocoa to create a window context with size, position, and pixel format.

Window Lifecycle

The windowing system handles opening, resizing, minimizing, and closing the window, along with managing event queues for keyboard, mouse, and other input.

GPU Integration

Windowing provides a drawable surface, such as a framebuffer or swapchain, which the GPU renders into using APIs like OpenGL, Vulkan, or DirectX.

Buffer Swapping

Buffer swapping swaps the back buffer (off-screen) with the front buffer (display) to present rendered frames smoothly and avoid flickering.

Event Handling

Event polling and callbacks let applications respond asynchronously to user input and window state changes, essential for interactivity.

Buffer Objects

      


        

          

            

              
Buffer Objects Overview

              

                Buffer objects are memory containers stored on the GPU, used to efficiently transfer large sets of data like vertices or indices from the CPU. They enable fast, reusable rendering workflows by avoiding per-frame data uploads.
              

            

          

        


        

          

            

              
Vertex Array Object

              

                A Vertex Array Object (VAO) stores the configuration for how vertex data is interpretedâ€”such as attribute pointers and buffer bindings. It simplifies switching between different mesh layouts by encapsulating this setup.
              

            

          

        


        

          

            

              
Vertex Buffer Object

              

                A Vertex Buffer Object (VBO) holds vertex data like positions, colors, and normals in GPU memory. Itâ€™s the core data structure used to define the shape and appearance of 3D geometry for rendering.
              

            

          

        


        

          

            

              
Index/Element Buffer Object

              

                An Element Buffer Object (EBO), also known as an Index Buffer Object (IBO), stores indices that reference vertices in a VBO. This allows for efficient reuse of vertices when rendering shapes like triangles or quads.
